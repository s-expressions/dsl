# S-expression standard: Personal name

```Lisp
(person
 (last-name "John")
 (other-names "P. Q. Public"))

(person
 (last-name "von Neumann")
 (other-names "John")
 (sort-key "von Neumann"))

(person
 (last-name "von Ettingshausen")
 (other-names "Andreas" "Freiherr")
 (sort-key "Ettingshausen"))

(person
 (full-name "鮑岳橋")
 (sort-key "Yueqiao"))
```
